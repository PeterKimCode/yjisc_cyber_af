Season 1. Digital Class Design 10 Lectures

Learn successful lecture operation strategies on the digital campus by following a curriculum designed like a journey.

Lesson 01

Digital Campus Orientation

Understand the GTCC Cyber Campus learning environment and explore learning management tools.

  • Keywords: Learning Space Setup, Creating Learning Routine
  • Task: Create My Weekly Learning Plan
Lesson 02

Learner Persona Analysis

Learn how to define data-based learner personas and reflect them in class design.

  • Keywords: Learning Data, User Journey
  • Task: Create Target Learner Persona Canvas
Lesson 03

Microlearning Design

Master strategies to segment content into short learning units and increase concentration.

  • Keywords: Learning Module, Cognitive Load Management
  • Task: Create 15-min Learning Module Storyboard
Lesson 04

Interaction Design

Cover interaction elements and real-time feedback strategies to increase learning immersion.

  • Keywords: Interactive Quest, Real-time Feedback
  • Task: Design Participatory Quiz Scenario
Lesson 05

Multimedia Production Guide

Share practical tips to improve lecture quality by integrating video, audio, and graphic elements.

  • Keywords: Storytelling, Branding Tone
  • Task: Create Filming Script and B-roll List
Lesson 06

Collaborative Task Design

Cover methods to design team projects enabling close collaboration online.

  • Keywords: Collaboration Tools, Team Learning
  • Task: Create Project Progress Roadmap
Lesson 07

Learning Analytics and Feedback

Practice interpreting learning data and providing personalized feedback.

  • Keywords: Learning Analytics Dashboard, Feedback Loop
  • Task: Configure Data-based Report Template
Lesson 08

Learner Support System

Introduce strategies for designing learning support services such as counseling, mentoring, and tutoring.

  • Keywords: Learning Coaching, Support Channels
  • Task: Create Support Service Flowchart
Lesson 09

Quality Assurance and Improvement

Design a QA process to continuously check and improve lecture quality.

  • Keywords: QA Checklist, Improvement Cycle
  • Task: Create Quality Inspection Rubric
